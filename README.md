 🌌 Xyro

🧩 Descripción

Xyro es una plataforma diseñada para ayudar a los estudiantes a descubrir qué carrera se adapta mejor a sus gustos, habilidades e intereses. Permite realizar pruebas vocacionales, explorar carreras, conocer universidades y recibir orientación académica.



🎯 Objetivo

Ayudar a los estudiantes a tomar mejores decisiones sobre su futuro académico y profesional.

 ⚡ Funcionalidades

|🧠 Test vocacional inteligente.
|🎓 Recomendación de carreras.
| 📖 Información sobre carreras.
|🏫 Universidades disponibles.
|⭐ Guardar carreras favoritas.
